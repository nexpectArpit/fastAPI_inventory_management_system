#use of dependcy injection
Dependency injection -> matlab aap function ke andar resource nahi banate, FastAPI aapko ready resource deta hai.
reason to use->
• Aap har route me database session banana nahi chahte.
• Bar-bar code repeat hota hai.
Example, galat tarika->
@app.get("/products")
def get_products():
    db = session()
    result = db.query(Product).all()
    db.close()
    return result

Issue->
Isse har route me
• session create
• session close
bar-bar likhna padega.

Solution->
Ek helper function banao jo session bana de.
FastAPI ko bolo ki jab koi route chale to wo session inject kare.
code ->
def get_db():
    db = session()
    try:
        yield db
    finally:
        db.close()


Unpacking ka simple meaning-> Dictionary ke key value pairs ko alag alag arguments ki tarah pass karna.
